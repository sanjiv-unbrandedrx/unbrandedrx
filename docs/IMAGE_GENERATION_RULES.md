# UnbrandedRx Image Generation Rules

## CRITICAL RULES - NEVER BREAK THESE

### 1. Bottle Label Text - ONLY THREE ELEMENTS
The bottle label must contain **exactly** these three text elements and **nothing else**:

1. **"unbrandedRx"** - Logo/brand name (black text, "Rx" as superscript)
2. **"[Active Ingredient]"** - Product name (e.g., "Enclomiphene Citrate")
3. **"RX ONLY"** - Small text at bottom

**NEVER include:**
- Description text ("Supports natural testosterone...")
- Marketing copy
- Dosage information
- Any other text
- Gibberish or placeholder text

### 2. No Lifestyle Photography
- NO smiling models
- NO couples
- NO faces
- NO full bodies
- NO beach/lifestyle scenes
- NO gym scenes

### 3. Hands Must Be Masculine
- Strong, confident hands
- Short, natural nails
- No nail polish
- Not overly veiny
- Neutral skin tones

### 4. Product is Always the Hero
- Product should be the focal point
- Clean, uncluttered compositions
- Minimal props/context

---

## Bottle Specification

```
Type: HDPE cylindrical pharmaceutical jar
Color: Matte white (body and cap)
Cap: Ribbed screw-top, white
Size: Standard supplement bottle (~120cc)
Label: White, minimal, clean typography
```

### Label Layout (top to bottom):
```
┌─────────────────────┐
│                     │
│    unbrandedRx      │  ← Logo, black, sans-serif
│                     │
│  Enclomiphene       │  ← Product name, black
│     Citrate         │
│                     │
│                     │
│      RX ONLY        │  ← Small, bottom, black
│                     │
└─────────────────────┘
```
